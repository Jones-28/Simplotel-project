# Simplotel-project
This project is a frontend web application built as part of a technical assignment.  
It is developed using **plain HTML and CSS**, without any external libraries or frameworks.  
The main goal of this project is to replicate a given UI design and make it **responsive** for desktop, tablet, and mobile screens.

---

## Features

- Responsive layout for all devices (desktop, tablet, mobile)
- Built entirely using HTML and CSS
- Pixel-perfect UI implementation
- Includes navigation bar, hero section, and content grids
- Clean and organized code structure

---

## Technologies Used

- HTML5
- CSS3

---

## Project Structure## How to Run the Project

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. View the fully responsive design
